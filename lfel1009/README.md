# [LFEL1009 - Getting Started with OpenTofu](https://training.linuxfoundation.org/express-learning/getting-started-with-opentofu-lfel1009/)


## OpenTofu Installation
``` shell
# 1. Download the Installer script:
curl -fsSL https://get.opentofu.org/install-opentofu.sh -o install-opentofu.sh

# 2. Grant execute permissions and review the script:
chmod +x install-opentofu.sh && less install-opentofu.sh

# 3. Install using the script:
./install-opentofu.sh --install-method standalone
```
